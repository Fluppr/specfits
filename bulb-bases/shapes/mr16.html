MR16 reflector shape

MR16 describes both a shape code (2-inch multifaceted reflector) and a base system. The shape is 50mm in diameter and mounts on a GU5.3 12V bi-pin base.

Exact specs

Shape codeMR16 (Multifaceted Reflector, 2-inch)
Maximum diameter2.0 in (50 mm)
Length (approximate)1.8 in (46 mm)
Base typeGU5.3 (12V low-voltage bi-pin)
Beam typeSpot or narrow flood

Shape vs base distinction

MR16 is simultaneously the shape name and the system name. When someone says "MR16 bulb" they mean both the 50mm reflector shape and the GU5.3 12V pin base together. A GU10 can also come in an MR16 shape — so the shape alone does not tell you the voltage.
