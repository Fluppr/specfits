Battery code lookup

Type any code printed on the battery itself — the canonical name, an alias, or even a partial code. Results update as you type.

Where to find the code: It is printed on the battery label or embossed on the metal casing. Common formats: CR2032, LR6, AG13, 2025, brown tab (hearing aid). If the old battery is gone, check the sticker inside the battery compartment or the original device manual.
