SR44 battery reference

An SR44 is a 1.55V silver-oxide button cell with the same physical size as LR44.

Exact specs

TypeSR44 (common aliases: 357, 303)
Diameter11.6 mm
Thickness5.4 mm
Nominal voltage1.55V
ChemistrySilver oxide
Typical capacity140-200 mAh (drain dependent)

View on Amazon

What devices commonly use it

Analog watches, camera accessories, and precision meters often use SR44 for stable output.

What it's interchangeable with

SR44 can replace LR44 in many devices of the same size, usually with better voltage stability over discharge.

What it's NOT interchangeable with

It should not be treated as equivalent to 3V lithium coin cells like CR-series batteries.

Common mistakes

  • Buying LR44 when a watch requires silver-oxide behavior.
  • Assuming the same alias code applies to all brands.
  • Overlooking device notes about low-drain vs high-drain cells.
